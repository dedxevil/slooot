Add Page Numbers to PDF

Drag & drop your PDF here, or click to select a file

How to Use the Page Numbering Tool

  • Select a PDF file you want to number.
  • Choose the position: top or bottom of each page.
  • Click Add Page Numbers and the modified PDF will download automatically.
  • Need to combine PDFs first? Use our PDF Merger.
  • Want to split a PDF into individual pages? Try the PDF Splitter.
  • Locked PDF? Use our Password Remover.

This tool works entirely in your browser. Your file is never uploaded to any server, ensuring full privacy and security.
